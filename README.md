# Cineworld Registration Form

Адаптивная форма регистрации для портала Cineworld с поддержкой светлой и тёмной тем.

## Функциональность

- ✅ Адаптивный дизайн
- ✅ Светлая и тёмная темы
- ✅ Валидация полей формы
- ✅ Календарь для выбора даты рождения
- ✅ Сохранение темы в localStorage
- ✅ Анимации и плавные переходы

## Технологии

- HTML5
- CSS3 (CSS Variables, Flexbox)
- JavaScript (ES6+)
- Air Datepicker (для календаря)
- Методология БЭМ

## Запуск проекта

### Способ 1: Открыть в браузере

1. Скачайте все файлы проекта
2. Откройте файл `index.html` в браузере

### Способ 2: Использовать Live Server

1. Установите расширение Live Server для VS Code
2. Откройте папку проекта в VS Code
3. Правой кнопкой на `index.html` → "Open with Live Server"
